# Google-Maps-API-Scanner

Check if the leaked Google-Maps API key is vulnerable or not.

# Usage 

  1. Clone the Repo 
  2. Paste API key you want to test.

# API's Checked 
 - Staticmap API
 - Streetview API
 - Embed (Basic-Free) API
 - Embed (Advanced-Paid) API
 - Directions API
 - Geocode API
 - Distance Matrix API
 - Find Place From Text API
 - Autocomplete API
 - Elevation API
 - Timezone API
 - Roads API
